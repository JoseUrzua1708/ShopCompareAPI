Documentación ShopCompare

¿Qué es ShopCompare?

ShopCompare es una API REST diseñada para centralizar la información de productos. Permite realizar comparaciones de precios en tiempo real entre las siguientes tiendas disponibles

Tiendas Disponibles

  • Bodega Aurrera
  • Chedraui
  • Soriana

Datos Disponibles

  • Tienda y Sucursal: Identificación del comercio y su ubicación física.
  • Producto: Nombre y descripción detallada del artículo.
  • Precio: Valor actualizado al momento de la consulta.
  • Timestamp: Hora exacta de la última actualización de datos.

Método de Conexión

Utiliza peticiones GET y POST enviando API Key en el header de autorización.
